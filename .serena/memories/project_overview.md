# Project Overview

## Purpose
Venturo Skeleton React is a comprehensive React + TypeScript dashboard skeleton/starter template built with Vite. It provides a production-ready foundation for building admin dashboards and web applications with multiple authentication options, theming, and internationalization support.

## Tech Stack

### Core
- **React**: v19.0.0
- **TypeScript**: v5.5.4 (strict mode enabled)
- **Vite**: v5.4.11 (build tool & dev server)
- **React Router**: v7.0.2

### Styling & UI
- **Tailwind CSS**: v3.4.16 (utility-first CSS)
- **Flowbite React**: v0.10.2 (primary component library)
- **Shadcn UI**: Additional components
- **Headless UI**: Unstyled primitives
- **Venturo UI**: Custom component library
- **Material UI**: v7.3.2
- **Lucide React**: Icons
- **Tabler Icons**: Icons

### State & Data
- **SWR**: v2.3.3 (data fetching & caching)
- **React Hook Form**: v7.62.0 (form management)
- **Axios**: v1.12.2 (HTTP client)

### Authentication
- **Firebase**: v11.6.0
- **Supabase**: v2.49.4
- **Auth0 SPA JS**: v2.1.3
- Custom JWT implementation

### Internationalization
- **i18next**: v23.5.1
- **react-i18next**: v13.2.2
- Languages: EN, AR, CH, FR
- RTL support included

### Other Libraries
- **MSW**: v2.7.3 (API mocking)
- **Moment.js**: Date handling
- **Lodash**: Utilities
- **SimpleMCP**: Scrollbar
- **Class Variance Authority**: Component variants

## Project Type
**Dashboard Skeleton / Starter Template** - Designed to be forked and customized for building admin panels, dashboards, and business applications.

## Key Features
1. Multiple authentication providers (Firebase, Supabase, Auth0, JWT)
2. Feature-based architecture for scalability
3. Dark/Light theme with customizer
4. Horizontal/Vertical layout modes
5. Multiple UI component libraries
6. i18n with RTL support
7. API client with automatic auth token injection
8. Lazy-loaded routes
